# Basic Attention Token

Basic Attention Token是由Javascript创始人为数字广告行业创造的新令牌。它为发布商的内容和用户的注意力付费，同时为广告商提供更多的广告回报。

Basic Attention Token与用户分享其广告收入的70%。只需[下载Brave浏览器](https://brave.com/)，在浏览器中启用"勇敢的广告"，并在浏览网页时获得有关广告的通知，即可获得免费的BAT。您还可以通过申请价值25-40 BAT的随机代币赠款来获得BAT。新的代币赠款通常每月提供一次。用户可以每30天申请一次代币授权，先到先得。您可以通过[下载Brave浏览器](https://brave.com/)并启用"Brave Rewards"来获得可能的代币奖励。当资助可用时，奖励图标上将显示一条通知。基本注意力令牌已经在[Coinmarketcap](https://coinmarketcap.com/currencies/basic-attention-token/)上列出。

# 在上网时获得免费 BAT（仅限 macOS、Windows 和 Linux 桌面设备）

1. 在您的桌面设备上下载并安装[Brave浏览器](https://brave.com/)。Brave Ads将在未来几个月内集成到Android和iOS应用程序中。

2. 启动Brave浏览器，然后单击地址栏旁边的BAT三角形徽标。

3. 将出现一个弹出窗口，您可以通过单击"加入奖励"来启用奖励。

4. 现在，从弹出窗口或键入"brave://rewards/"导航到您的奖励设置，并确保"Brave Ads"开关已打开。

5. 从现在开始，您将在浏览器认为适当且不具有破坏性的时间以通知的形式向您提供优惠。

6. 只需参与这些通知，查看广告，您就会从观看的广告中获得70%的收入份额，而Brave只会收到剩余的30%。

7. 在勇敢奖励每月周期结束时，您可以申领累积的代币。BAT可以每月自动捐赠给您最常访问的网站，或用于向喜爱的内容创作者或朋友提供小费。很快，您还可以将BAT用于优质内容，并将其兑换为现实世界的奖励，例如酒店住宿，餐厅优惠券和礼品卡。Brave还在研究一种选择，让用户从钱包中提取BAT供个人使用，通过交换合作伙伴将他们的BAT转换为当地的法定货币。

8. 请注意，Brave Ads 的地理位置仅限于以下国家/地区：美国、英国、加拿大、德国和法国。更多地区将很快推出。

9. 您可以在其[官方公告](https://brave.com/brave-ads-launch/)中找到有关勇敢广告的更多信息。

# 桌面用户的每月令牌授予

1. [从这里下载](https://brave.com/)勇敢的浏览器 。

2. 运行浏览器并启用"勇敢的奖励"。从 Brave 浏览器界面中单击 BAT 图标。如果有可用的令牌授权，您将看到相应的消息，并可以通过单击"确定"开始声明您的BAT令牌。

3. 之后，单击"奖励设置"，然后单击"索赔"按钮并完成一个简单的验证码拼图，以完成您的索赔。

4. 新的代币赠款通常每月提供一次。用户可以每30天申请一次代币授权，先到先得。当免费令牌授予可用时，奖励图标上将显示一条通知。

5. 如果您在申领免费代币时遇到问题，可以查看此[官方视频教程](https://www.youtube.com/watch?v=Jcbu5Im_FvQ)，[以获取免费的BAT代币授予](https://www.youtube.com/watch?v=Jcbu5Im_FvQ)。

6. 代币不能直接提取，但收到的BAT可以用来给某人（任何朋友或你想支持的内容创建者）小费。一旦获得小费，代币可由小费人/抵押人提取。

# Android/iOS 应用用户的每月代币授予（25 – 40 BAT）

1. 在您的安卓或 iOS 设备上下载 [Brave 浏览器](https://brave.com/)。

2. 转到 Brave 浏览器应用程序中的搜索栏，然后键入chrome://flags。

3. 搜索"奖励"，然后单击"默认"按钮，然后选择"已启用"。

4. 单击"立即重新启动"，完成后，Brave浏览器界面中将出现一个BAT图标。

5. 现在，您可以完成桌面步骤中提到的相同步骤，以有资格从价值 25-40 个 BAT 令牌的令牌授予中获取资格。
